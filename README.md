# Клуб четырех коней

Адаптивный лендинг шахматного турнира, выполненный по макету Figma в рамках тестового задания.

## Стек

- HTML5
- CSS3
- JavaScript
- Без библиотек и фреймворков

## Реализовано

- Адаптивная верстка для desktop, tablet и mobile
- Бегущая строка
- Якорные кнопки на первом экране
- Зацикленная карусель участников с автопереключением каждые 4 секунды
- Карусель этапов без зацикливания и без автопереключения
- Анимации появления контента и декоративных элементов
- SEO-метатеги, Open Graph и JSON-LD
- Поддержка `prefers-reduced-motion`

## Запуск

Открыть файл `index.html` в браузере.
